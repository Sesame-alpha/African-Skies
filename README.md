
# Airlines Africa
A browser-based aviation intelligence dashboard for African airline operations teams, airport authorities, and regulators.

## Features
- Live flight map with animated aircraft (Canvas + Mercator projection)
- Flight search and status filters
- METAR weather with impact scoring for GBE, JNB, ADD, NBO, CPT
- ML delay prediction powered by TensorFlow.js
- Active NOTAMs, UTC clock, and CSV export
- Fully mobile responsive

## Stack
HTML · CSS · Vanilla JavaScript · TensorFlow.js

## Usage
No setup required. Open `airlines-africa.html` in any modern browser.
